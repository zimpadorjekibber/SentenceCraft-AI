# **App Name**: SentenceCraft AI

## Core Features:

- Word Input Box: A user-friendly box to input words of various types (subject, object, verb, preposition, etc.).
- AI Sentence Generator: AI tool to generate grammatically correct and contextually relevant sentences using the words provided in the input box. Uses tense rules to construct the sentence.
- Generated Sentence Display: Display the generated sentence in a clearly visible area.
- Sentence modification selection: Display sentence options and allow for corrections or additional modifications
- Tense Selection and Generation: Display all 12 tenses in an image, and when the user clicks on any of the tenses displayed, the AI tool generates the sentences based on the selected tense. The tense rules are pre-existing in the tool.

## Style Guidelines:

- Primary color: HSL(210, 70%, 50%) which translates to a shade of blue, represented in hex as #3399FF, evoking feelings of intelligence, clarity, and trust.
- Background color: A very light shade of blue (HSL(210, 20%, 95%), hex: #F0F8FF), offering a clean and unobtrusive backdrop that is easy on the eyes.
- Accent color: Use a complementary orange color (HSL(30, 80%, 50%), hex: #FFA533) to highlight key elements such as the 'Generate' button, providing a clear call to action.
- Body and headline font: 'Inter', a sans-serif font, will be used for a modern, neutral, and highly readable experience.
- The layout should be clean and intuitive, with a clear separation between the input area, the AI processing section, and the display area.
- Use subtle animations for generating the sentence. It can provide engaging user feedback
- Incorporate simple, related icons to label different elements
